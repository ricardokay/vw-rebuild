A Text-forward No image area at all. 3px red left border. Italic PT Serif headline. Headline and byline carry the full weight. Same card class (.vw-card--text) already live in Zone D.
B Per-section art Image slot filled with CSS-generated art keyed to section. Echoes the geometric mark system (Direction A — Paired Bars). No stock photo, no lorem image. Signals section identity without a real photograph.
C Duotone block Image slot = solid #C41230 block with the headline overlaid in white. Bold editorial. Below the block: category + byline only. ⚠ Design brief constraint: "Never use [red] as a large background — accent only, not a ground." This direction requires an explicit decision to override that rule.
